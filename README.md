OpenCV 图像处理实验
实验目的
使用 C++/OpenCV 完成图像读取、信息输出、显示、灰度转换、保存及区域裁剪，掌握基础图像处理流程。
使用 Python/OpenCV 对比全局直方图均衡与 CLAHE 在 Moon/Coin 图上的增强效果，分析差异。
使用 OpenCV 对 3 幅不同类型图像（含低对比度 / 噪声）进行增强处理，对比多种算法效果，完成自定义实现与定量分析。
实验任务
一、基础图像处理
读取测试图片
输出图像尺寸、通道数、数据类型
显示原图
转换为灰度图并显示
保存灰度图
裁剪左上角区域、输出像素值
二、直方图均衡对比
安装 scikit-image，使用内置 Moon/Coin 图
用 OpenCV 实现全局直方图均衡与 CLAHE
对比两种方法效果，显示图像与直方图
三、图像增强综合实验
实现全局直方图均衡、CLAHE
实现均值滤波（自定义实现）、高斯滤波、中值滤波
实现拉普拉斯锐化
实现「滤波→均衡」与「均衡→滤波」组合处理
对比不同参数影响，展示结果与对应直方图，分析实验现象
给出至少两项定量评价指标（PSNR/SSIM）
环境要求
C++ 编译器（GCC/Clang/MinGW）、OpenCV 4.6、CMake 3.10+
Python 3.8+、NumPy、scikit-image、opencv-python、matplotlib
